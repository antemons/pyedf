# pyedf
The module glues Tuenis' edflib with python using ctypes.


INSTALLATION
----------------
In the head directory, enter the command 'make'.
Write into your '~/.bashrc' the line

> PYTHONPATH += <PATH TO HEAD DIRECTORY>:
> export PYTHONPATH

For example mine is

> PYTHONPATH = $PYTHONPATH:/home/jus/Code
